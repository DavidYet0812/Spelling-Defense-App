# 🧙‍♂️ 拼字防禦戰 (Spelling Defense)

一款結合**英文拼字學習**與**塔防遊戲**的互動式 Web 應用程式。看中文、拼英文，在遊戲中不知不覺背好單字！

## ✨ 功能特色

### 🎮 遊戲模式與學習流程

遊戲提供了完整的學習到測驗的流程，支援以下核心功能：

| 功能/模式 | 說明 |
|------|------|
| 📚 **分類選擇** | 開始遊戲或預覽前，玩家可選擇特定的單字分類（例如：日常生活、動物等）進行階段性學習 |
| 👀 **單字總覽** | 先背後考！進入圖鑑模式觀看該分類底下的所有單字與中文翻譯 |
| 📖 **練習模式** | 一次出現一個中文提示，不限時間、不會扣血，慢慢學習拼字，適合初學者 |
| ⚔️ **防禦模式** | 英文單字的中文翻譯從天而降，玩家必須在觸底前拼出正確英文來消除它 |

### 🎯 核心玩法

- **看中文，拼英文** — 畫面只顯示中文翻譯，玩家必須從記憶中回想英文拼法
- **即時回饋** — 拼對的字母以綠色顯示，未拼的以底線表示
- **難度遞增** — 每得 100 分，單字下落速度加快
- **🔥 狂暴模式** — 5 秒內連續消除 5 個單字即可觸發，持續 10 秒，速度減半、分數加倍

### 🎨 視覺效果

- **程式化星空背景** — 80 顆閃爍星星 + 隨機流星動畫
- **巫師角色** — Canvas 即時繪製，三種動態（待機浮動 / 施法發光 / 受傷閃爍）
- **消除特效六層疊加**：
  - 💥 三色粒子爆發（帶發光拖尾）
  - 🔵 衝擊波環形擴散
  - 🔤 字母旋轉飛散
  - ⭐ 四角星芒閃爍
  - 📈 分數飄字上浮（+10 / +20）
  - ⚡ 螢幕白色閃光

### 🔊 音效系統

全部使用 **Web Audio API** 程式化合成，零外部音效檔案：

| 音效 | 觸發時機 |
|------|----------|
| 🎵 背景音樂 | Am → F → C → G 和弦琶音循環 |
| ✅ 打字音 | 每拼對一個字母 |
| ✨ 消除音 | 完整拼出單字（上升音階 + 閃光音） |
| ❌ 錯誤音 | 拼錯字母 |
| 💥 受傷音 | 單字觸底扣血 |
| 🔥 狂暴音 | 狂暴模式啟動 |
| 💀 結束音 | 遊戲結束（下降悲傷音階） |

音效與背景音樂各有**獨立開關**，可隨時靜音。

## 🚀 快速開始

### 直接開啟

```bash
# 方法一：直接用瀏覽器開啟
open index.html

# 方法二：用任何靜態伺服器
npx serve .
# 或
python -m http.server 8000
```

### 設定後端（選用）

1. 開啟 [Google Apps Script](https://script.google.com/)
2. 建立新專案，實作 `doGet()` 和 `doPost()` 端點
3. 發布為 Web App（存取權限設為「所有人」）
4. 將部署網址填入 `api.js` 中的 `GAS_URL`

> 若不設定 GAS，遊戲會自動使用內建的 6 個預設單字。

## 🎮 操作說明

### 遊戲操作

| 操作 | 方式 |
|------|------|
| 輸入字母 | 點擊虛擬鍵盤 或 實體鍵盤 |
| 音效開關 | 點擊狀態列 🔊 按鈕 |
| 背景音樂 | 點擊狀態列 🎵 按鈕 |
| 離開練習 | 按 `ESC` 鍵 |

### 防禦模式規則

- ❤️ 初始 3 條生命
- 單字觸底 → 扣 1 HP + 畫面震動
- HP 歸零 → 遊戲結束，可記錄排行榜
- 一般消除 → +10 分
- 狂暴模式消除 → +20 分

### 練習模式規則

- 📖 無生命值限制
- 一次一題，拼完自動換新題
- 題庫不重複（用完重新循環）
- 即時顯示正確率統計

### 單字總覽 (圖鑑模式)

- 👀 直接瀏覽選擇的分類下的所有單字清單
- 卡片化呈現英文與中文翻譯，無壓力的考前記憶

## 📋 教師管理後台

存取 `admin.html` 可進入教師管理介面：

- **CSV 大量匯入** — 上傳格式：`英文單字,中文翻譯,分類(選填),難度(選填)`
- **即時預覽** — 上傳前可檢視所有資料，自動標記格式錯誤
- **資料驗證** — 自動略過缺少必填欄位的無效資料

## 🔧 自訂設定

### 修改預設題庫

編輯 `api.js` 中的 `fallbackVocab` 陣列：

```javascript
const fallbackVocab = [
    { word: "apple", translation: "蘋果" },
    { word: "banana", translation: "香蕉" },
    // 新增更多單字...
];
```

### 調整遊戲難度

編輯 `game.js` 中的 `gameState`：

```javascript
const gameState = {
    hp: 3,              // 初始生命值
    baseSpeed: 1.0,     // 基礎下落速度
    spawnInterval: 2000  // 生成間隔（毫秒）
};
```
